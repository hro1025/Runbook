# Runbook

Runbook is a terminal-based script management tool. You can execute, edit, create, and delete scripts from a simple TUI interface.

## Scripts Location
Scripts are stored in `~/.scripts/runbook/`

## Keybinds

### Normal Mode
| Key     | Action |
|---------|--------|
| `Enter` | Run script |
| `E`     | Edit script |
| `C`     | Create script |
| `D`     | Delete script |
| `Esc`   | Quit |

### Edit Mode
| Key      | Action |
|----------|--------|
| `Ctrl+S` | Save |
| `Esc`    | Cancel |
